==================================================
БИБЛИОТЕЧНАЯ СИСТЕМА - ИНСТРУКЦИЯ ПО ЗАПУСКУ
==================================================

Файлы проекта:
- main.py              # Основное приложение FastAPI
- models.py            # Модели данных (20 таблиц)
- database.py          # Подключение к PostgreSQL
- requests.py          # Функции для работы с БД
- create_tables.py     # Создание таблиц
- fill_data.py         # Заполнение тестовыми данными
- drop_tables.py       # Удаление таблиц (при необходимости)
- simple_api_test.py   # Тест API

==================================================
ШАГ 1: УСТАНОВКА PYTHON
==================================================

Если Python не установлен:
1. Скачайте с https://www.python.org/downloads/ (версия 3.10 или новее)
2. При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"

Проверка установки (в командной строке):
python --version
# Должна появиться версия Python

==================================================
ШАГ 2: УСТАНОВКА БИБЛИОТЕК
==================================================

Установите Visual Studio Code: https://code.visualstudio.com/download

Откройте папку с проектом в Visual Studio Code
Создайте новый терминал в папке с проектом.

Напишите в нём одну из команд:

Вариант A (основной):
pip install fastapi sqlmodel uvicorn psycopg2-binary

Вариант B (если не работает pip):
python -m pip install fastapi sqlmodel uvicorn psycopg2-binary

Вариант C (для Linux/Mac или нескольких Python):
pip3 install fastapi sqlmodel uvicorn psycopg2-binary

Проверка установки:
pip list
# Должны быть: fastapi, sqlmodel, uvicorn, psycopg2-binary

==================================================
ШАГ 3: ПОДГОТОВКА БАЗЫ ДАННЫХ
==================================================

Проект использует удалённую БД PostgreSQL.
Подключение уже настроено в database.py.

0. Если вы не впервые создаёте таблицы/не уверены, 
что схема пустая, то удалите таблицы:
python create_tables.py

1. Создайте таблицы:
python drop_tables.py

2. Заполните тестовыми данными:
python fill_data.py

При успешном выполнении увидите:
✅ Все справочники заполнены!
✅ Создано 5 читателей
✅ Создано 6 книг
✅ И т.д.

==================================================
ШАГ 4: ЗАПУСК СЕРВЕРА
==================================================

Запустите сервер API:

python -m uvicorn main:app --reload


При успешном запуске увидите:
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.

==================================================
ШАГ 5: ПРОВЕРКА РАБОТЫ
==================================================

1. Откройте браузер и перейдите по адресу:
http://localhost:8000

2. Документация API (Swagger):
http://localhost:8000/docs
Можете проверить различные запросы (GET, POST, PUT, DELETE)

3. Альтернативная документация (ReDoc):
http://localhost:8000/redoc

3. Запустите тестовый скрипт (проверяет все endpoints):
python simple_api_test.py
python test_db.py


==================================================
ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
==================================================

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'fastapi'"
РЕШЕНИЕ: Выполните ШАГ 2 (установка библиотек)

ПРОБЛЕМА: "Connection refused" при запуске
РЕШЕНИЕ: Убедитесь, что порт 8000 свободен:
netstat -ano | findstr :8000

ПРОБЛЕМА: Ошибка подключения к БД
РЕШЕНИЕ: Проверьте интернет-соединение
         БД находится на удалённом сервере 176.108.247.125

ПРОБЛЕМА: "Address already in use"
РЕШЕНИЕ: Измените порт (например, на 8001):
uvicorn main:app --reload --port 8001
